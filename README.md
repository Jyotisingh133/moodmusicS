------------------------------------------------------------------------

# 📖 About the Project

MoodMusicS is a web‑based music recommendation system designed to
provide songs that match a user's emotional state.

Music has a strong connection with human emotions. This application
allows users to quickly discover songs that align with their mood such
as:

-   Happy 😊
-   Sad 😔
-   Relaxed 😌
-   Energetic ⚡
-   Romantic ❤️

The goal of this project is to create a simple and interactive music
platform where users can instantly enjoy songs that match their
feelings.

# 🧠 How It Works

1.  User opens the web application\
2.  User selects their current mood\
3.  System processes the mood\
4.  Matching songs or playlists are displayed\
5.  User plays and enjoys the music

# ⚙️ Installation

Clone the repository

    git clone https://github.com/Jyotisingh133/moodmusicS.git

Go to project folder

    cd moodmusicS

Run Spring Boot

    mvn spring-boot:run

Open in browser

    http://localhost:8080
